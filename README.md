### Как теперь "спрятать" venv от Гита:

Чтобы Git перестал видеть папку `venv` и предлагать её закоммитить, тебе нужен файл `.gitignore`.

1. Создай в главной папке файл с названием **`.gitignore`** (точка в начале обязательна).
2. Впиши туда просто одну строку:
   ```text
   venv/
   .env
   __pycache__/
   *.db
